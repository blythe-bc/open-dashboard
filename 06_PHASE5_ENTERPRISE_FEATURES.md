# Phase 5 — Enterprise Features (선택)
목표: 조직 확장/타 시스템 연동/규정 준수 수준 강화.

---

## 옵션 기능(우선순위 순)
1) Export (Excel/PDF) + 워터마크/권한 반영
2) Schedule / Subscription (이메일/Teams 알림 등)
3) Embed(내부 포털 iframe) + 권한 연동
4) Object-level 권한 강화(Dataset/Metric/Dashboard/Widget)
5) 고급 캐시 전략(프리페치/증분 갱신)
6) 멀티 테넌시/멀티 DB (필요 시)

---

## Acceptance Criteria
- 대규모 사용자/대규모 데이터에서도 운영 가능
- 규정 준수(감사/권한/마스킹) 강화
